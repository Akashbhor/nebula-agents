# React Frontend Stack Pack

Use this pack for React + TypeScript + Vite products.

## Stack Profile

- Framework: React 18+
- Build: Vite
- Styling: Tailwind CSS or the product-standard styling system
- Components: shadcn/ui or product-standard component library
- Routing: React Router or the product-standard router
- Server State: TanStack Query or the product-standard client cache
- Forms: React Hook Form + AJV or the product-standard form stack
- Testing: Vitest, React Testing Library, jest-axe

## Working Rules

- Keep feature UI co-located.
- Keep API wiring explicit and type-safe.
- Keep accessibility and theme checks in the same slice.
- Keep behavior tests with the feature that changed.
