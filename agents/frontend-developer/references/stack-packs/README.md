# Frontend Stack Packs

These files hold stack-specific frontend guidance that stays out of the core `frontend-developer` skill.

## Selection Rule

- Use the `React` pack for React + TypeScript + Vite products.
- Use the `Python / Streamlit` pack for Streamlit products.
- Use the `htmx` pack for server-driven HTML products.
- Do not load multiple frontend packs for the same session unless the product actually contains multiple frontend stacks.

## Packs

- `react/pack.md`
- `streamlit/pack.md`
- `htmx/pack.md`
